# AUCOSMOS — Organ Days Quiet-Room Update

Bismillah.

This version turns The Organ Days into a true doorway experience:

- `organ-days.html` is the entry room.
- Each organ has its own quiet page.
- Individual organ pages no longer have a full top navigation bar.
- Each organ page ends with one small link: **Return to the doors**.
- Door links use a subtle fade-to-dark transition before the next room opens.
- Heart Day remains isolated, with the optional soft heartbeat button and final coda.
